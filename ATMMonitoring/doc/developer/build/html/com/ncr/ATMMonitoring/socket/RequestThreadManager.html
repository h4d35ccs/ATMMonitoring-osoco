RequestThreadManager¶

public class RequestThreadManager extends Thread¶

The Class RequestThreadManager. The class in charge of distributing the pending to request data ip’s among multiple RequestThread’s.

Author :Jorge López Fernández (lopez.fernandez.jorge@gmail.com)

RequestThreadManager¶

public RequestThreadManager(double maxThreads, double maxTerminals, int timeOut, int agentPort, int sleepTime, int maxTime, SocketService socketService, Set<String> ips)¶

Instantiates a new request thread manager.

Parameters:
  • maxThreads – the max number of threads
  • maxTerminals – the max number of terminals per thread
  • timeOut – the response time out
  • agentPort – the agent port
  • sleepTime – the sleep time between checks onto the sub-threads
  • maxTime – the max time we wait for the whole process to end
  • socketService – the socket service
  • ips – the ips
