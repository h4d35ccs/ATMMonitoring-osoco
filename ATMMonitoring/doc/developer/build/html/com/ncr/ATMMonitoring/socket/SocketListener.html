SocketListener¶

public class SocketListener extends Thread¶

The listener interface for receiving socket events. The class that is interested in processing a socket event implements this interface, and the object created with that class is registered with a component using the component’s addSocketListener method. When the socket event occurs, that object’s appropriate method is invoked.
